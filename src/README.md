# Todo App

A Spring Boot REST API for managing todos with JWT authentication and role-based authorization.

## Features

- 1:N relationship between users and todos
- JWT-based authentication
- Role-based access control (ADMIN/USER)
- CRUD operations for todos
- H2 database
- BCrypt password encoding

## Technológiák

- Java
- Spring Boot
- Spring Security
- Spring Data JPA
- H2 Database
- JWT (JSON Web Tokens)
- Maven
